# Ravens Hollow: A Text Adventure Game

## How to Run:
1. Download the game files to your computer.
2. Make sure you have Python 3 installed.
3. Install any necessary dependencies (such as `curses` and `playsound` if required).
4. In your terminal, navigate to the folder where the game is located.
5. Run the game by executing the following command:
   python game.py

## Requirements:
- Python 3.x
- Curses library (usually pre-installed with Python on Unix-based systems).
- Optional: Playsound library for sound effects (use pip to install: `pip install playsound`).

## Gameplay:
1. The game starts with a menu where you can choose to start the game, read the lore, or exit.
2. You will explore different areas like the ruined theater and encounter puzzles, jumpscares, and mysterious events.
3. Your choices will determine which ending you get.
